Installation Instructions

Note: Since these APKs are not from the Play Store, your device will ask for permission to install. This is normal for all directly-distributed Android apps.

1. Allow installation from unknown sources

  • Android 8+: Settings → Apps → Special App Access → Install Unknown Apps → find your browser → Allow.
  • Android 7 and below: Settings → Security → enable Unknown Sources.

2. Google Play Protect warning

You may see "App not recognized". Tap More details then Install anyway. The app is safe.

3. Install

  1. Download the APK file from above.
  2. Open the downloaded file from your notifications or file manager.
  3. Follow the on-screen prompts to install.
  4. Open Vehium from your app drawer.
Note: Windows SmartScreen may show a warning because the app is distributed outside the Microsoft Store. This is normal.

1. SmartScreen warning

  1. When the blue SmartScreen dialog appears, click More info.
  2. Then click Run anyway.

2. Install

  1. Download the installer from above.
  2. Double-click the downloaded .exe file.
  3. Follow the installation wizard.
  4. Open Vehium Workshop from your desktop or Start menu.
  5. Log in with your workshop credentials.

1. Install TestFlight

Download TestFlight from the App Store (free).

2. Accept the invitation

Open the invitation link sent by the Vehium team on your iPhone or iPad. Accept the invite and install through TestFlight.

3. Updates

When a new version is available, TestFlight will notify you. Open TestFlight and tap Update.

TestFlight builds expire after 90 days. If the app stops working, request a new build from the team.
