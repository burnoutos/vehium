# Vehium – Windows Desktop App (Remote Workshop)

> **Vehium is a Garage Management System that allows workshops to manage jobs and notify customers in real time about the status of their vehicle.**

> **This is a standalone `.exe` installer. It is not distributed via the Microsoft Store.**

---

## ⚠️ Before You Install – Please Read

### 1. Windows SmartScreen Warning

Because this application is distributed outside the Microsoft Store, Windows SmartScreen may display a warning when you run the installer. This is expected behaviour for independently distributed software.

To proceed:

1. When the blue SmartScreen dialog appears, click **More info**.
2. Then click **Run anyway**.
3. The installation will continue normally.

### 2. No Automatic Updates

This desktop app does **not update itself automatically**. To get a newer version:

- Come back to this page periodically.
- Download the latest `.exe` installer.
- Run it over the existing installation — your data and settings will be preserved.

---

## 📦 Installation Steps

1. Download the `vehium-workshop-setup.exe` file from this page.
2. Double-click the downloaded file to launch the installer.
3. If prompted by SmartScreen, click **More info → Run anyway**.
4. Follow the on-screen installation steps.
5. Once installed, open **Vehium Workshop** from your desktop or Start menu.
6. Log in with your workshop credentials provided during onboarding.

---

## 📋 Requirements

- Windows **10** or **11** (64-bit)
- ~200 MB free storage
- Internet connection required
- Active workshop account on Vehium
