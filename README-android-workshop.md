# Vehium – Android APK (Workshop)

> **Vehium is a Garage Management System that allows workshops to manage jobs and notify customers in real time about the status of their vehicle.**

> **This is a direct (sideloaded) APK release. It is not distributed via the Google Play Store.**

---

## ⚠️ Before You Install – Please Read

### 1. Allow Installation from Unknown Sources

Since this APK is not from the Play Store, Android will block it by default. You need to allow installation manually:

- **Android 8+**: Go to **Settings → Apps → Special App Access → Install Unknown Apps**, find your browser or file manager, and toggle **Allow from this source**.
- **Android 7 and below**: Go to **Settings → Security** and enable **Unknown Sources**.

### 2. You May See a Security Warning or Scan Prompt

Google Play Protect may flag the APK with a warning like _"App not recognized"_ or ask to scan it. This is expected for any APK installed outside the Play Store.

To proceed:

1. Tap **More details** (or **Install anyway**) when the warning appears.
2. If Play Protect asks to scan — you can allow the scan or tap **Install without scanning**. The app is safe.

### 3. No Automatic Updates

Because this is a raw APK and not a Play Store release, **the app will not update itself automatically**. To get a newer version:

- Come back to this page periodically.
- Download the latest APK file manually.
- Install it over the existing app — your data will be preserved.

---

## 📦 Installation Steps

1. Download the `.apk` file from this page.
2. Open the downloaded file (via your notification bar or file manager).
3. Follow the on-screen prompts, granting install permission if asked.
4. Once installed, open **Vehium Workshop** from your app drawer.
5. Log in with your workshop credentials provided during onboarding.

## 📋 Requirements

- Android **7.0 (Nougat)** or higher
- ~120 MB free storage
- Active workshop account on Vehium
